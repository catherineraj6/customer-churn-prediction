# customer-churn-prediction

# Project Overview
In this project, the goal was to predict customer churn using a logistic regression model. The purpose of this is to find the customers that leave early, in order to come up with strategies to maintain retention.

# Installation Instructions
The libraries used for this project were kagglehub, numpy, pandas, matplotlib, seaborn, joblib and sklearn. Before importing the libraries in the Python notebook, please run the following code: !pip install kagglehub. Run the same command with the other libraries if needed. 

# Usage examples
A telecommunication company could use this model to predict how likely a customer is to churn. The company could use the results of the model to come up the retention efforts as needed. For example, customers with high monthly charges are most likely to churn, so the business owners can consider lowering the costs or coming up with discounts.

# Key findings
Customers with month-to-month contracts are most likely to churn. 
Fiber optic service customers are likely to churn.
Customers with shorter tenure were more likely to leave.
High monthly charges were highly correlated with churn.
The logistic regression model achieved about 80% in accuracy
